A continuación se muestran las clases que ofrece el framework Bootstrap equivalentes a las propiedades CSS para la aplicación del metodo de diseño Flexbox. Dichas clases se aplican al elemento hijo.

Propiedad "order"

Es una propiedad de CSS, la cumple la función es reordenar los elementos HTML. Bootstrap ofrece para esta propiedad clases con 12 valores que van desde el 1 hasta el 12, por ejemplo, order-1, order-2, order-3, etc. El valor por defecto de order es cero.

Propiedad "flex-grow"

Es una propiedad de CSS Flexbox que permite que un elemento flexible crezca o no. Bootstrap ofrece para esta propiedad clases con 2 valores flex-grow-0 (el elemento no crece), flex-grow-1 (el elemento crece). El valor por defecto de flex-grow es cero.

Propiedad "flex-shrink"

Es una propiedad de CSS que permite indicar si un elemento flexible se va achicar o no. Todos los elementos en Flexbox cuenta con esta propiedad que por defecto tiene como valor uno. Esto significa que todos los elementos en el momento que no quepan en el elemento padre, se van a empezar achicar. Bootstrap ofrece para esta propiedad clases con 2 valores flex-shrink-0 (el elemento no se achica), flex-shrink-1 (el elemento se achica).

Propiedad "align-self"

Esta propiedad CSS de flexbox permite alinear los elementos en forma individual en el cross axis (eje vertical). Bootstrap ofrece para esta propiedad clases con 5 valores align-self-start, align-self-end, align-self-center, align-self-baseline, align-self-stretch. El valor por defecto de align-self es stretch.
