A continuacion se muestran las clases que ofrece el framework Bootstrap equivalentes a las propiedades CSS para la aplicación del método de diseño Flexbox. Dichas clases se aplican al elemento padre.

Propiedad "d-flex"

En la estructura inicial, las cajas aparecen apiladas una debajo de la otra. La razón, es porque la etiqueta que las contiene tiene la propiedad display con el valor block por defecto.

Al aplicar la propiedad "d-flex" al elemento contenedor padre, los elementos hijos se transforman en cajas flexibles. Cuando se utiliza flexbox en un elemento, este no se vuelve flexible, sino los elementos que estan contenidos en este.

Propiedad "flex-direction"

Permite modificar la dirección de las cajas flexibles que por defecto es horizontal. Existen 4 valores para esta propiedad que ofrece Bootstrap en clases: flex-row, flex-row-reverse, flex-column, flex-column-reverse. El valor por defecto es flex-row.

Propiedad "flex-wrap"

Esta propiedad permite indicar si se quiere envolver o no todos los elementos en una fila. Existen 3 valores para esta propiedad que ofrece Bootstrap en clases: flex-nowrap, flex-wrap, flex-wrap-reverse. El valor por defecto es flex-nowrap.

Propiedad "justify-content"

Modo horizontal

Esta propiedad permite centrar los elementos flexibles en el main-axis (eje principal) que por defecto se encuentra en modo horizontal. Tambien se puede implementar en modo vertical agregando, ademas, la clase flex-column Existen 5 valores para esta propiedad que ofrece Bootstrap en clases: justify-content-start, justify-content-end, justify-content-center, justify-content-between, justify-content-around. El valor por defecto es justify-content-start.

Propiedad "align-items"

Esta propiedad permite alinear los elementos en el cross axis (eje transversal) que por defecto es vertical, pero tambien se puede modificar para utilizar en horizontal agregando la clase flex-column. Existen 5 valores para esta propiedad que ofrece Bootstrap en clases: align-items-start, align-items-end, align-items-center, align-items-baseline, align-items-stretch. El valor por defecto es align-items-stretch.

Propiedad "align-content"

Esta propiedad permite alinear los elementos en el cross axis (eje transversal) en grupo. Esta es la diferencia con la propiedad "align-items", que alinea los elementos linea por linea. Existen 6 valores para esta propiedad que ofrece Bootstrap en clases: align-content-start, align-content-end, align-content-center, align-content-between, align-content-around, align-content-stretch. El valor por defecto es align-items-stretch.
